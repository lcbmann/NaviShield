Learn More - NaviShield

NaviShield uses a combination of AI-based phishing detection, Google Safe Browsing checks, and WHOIS domain registration data to help you determine if a URL might be malicious.

How It Works:

  1. URL Normalization: We ensure the URL is properly parsed and uses a valid scheme (HTTP/HTTPS).
  2. Safe Browsing Check: The URL is checked against Google’s Safe Browsing API, which flags known malicious sites.
  3. AI Analysis: If the URL isn’t already flagged, it's analyzed by an AI model trained to detect phishing patterns.
  4. WHOIS Lookups: We fetch domain registration data (e.g., creation date, registrar details, domain age). A brand-new or private-registered domain can be suspicious in some contexts.
  5. Final Scoring & Result: We combine all checks (Safe Browsing, AI model prediction, WHOIS info) to produce a “Safe,” “Phishing,” or “Uncertain” label. The domain’s WHOIS data can help identify unusually new or hidden registrations, raising the suspicion score.
