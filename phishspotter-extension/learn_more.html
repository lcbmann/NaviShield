About PhishSpotter

PhishSpotter uses a combination of AI-based phishing detection and Google Safe Browsing to help you determine if a URL is safe or potentially harmful.

How it Works:

  1. URL Normalization: The URL is standardized to ensure consistency.
  2. Safe Browsing Check: The URL is first checked against Google's Safe Browsing API.
  3. AI Analysis: If the URL passes Safe Browsing, it is then analyzed using an AI model.
  4. Final Result: Based on the results from both checks, the extension reports the URL as "Safe," "Phishing," or "Uncertain."
