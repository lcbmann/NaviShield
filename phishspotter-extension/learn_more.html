About PhishSpotter

PhishSpotter uses a combination of AI-based phishing detection and Google Safe Browsing to help you determine if a URL is safe.

How It Works:

  1. URL Normalization: The URL is standardized.
  2. Safe Browsing Check: Checked against Google’s Safe Browsing API.
  3. AI Analysis: If not flagged, the URL is analyzed by our model.
  4. Final Result: We combine both checks to label the URL “Safe,” “Phishing,” or “Uncertain.”
